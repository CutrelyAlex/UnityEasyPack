# UnityEasyPack
简单好用的Unity游戏功能包~
制作宗旨：“即插即用”，方便独立游戏开发者快速开发Demo或在正式项目中使用

## 使用方法
直接Clone到项目里即可，无需其他依赖。

## 目前支持
- 基于组件的**BUFF系统**
- 基于修饰器的**游戏属性**
- **组合式的属性**
- Inventory系统
- Gameplay的寻路
- "万物卡牌"

可以查看每个模块下的 Example 文件夹查看使用示例~
